Pour changer le modèle:

Dans TKinter_visualization.py:
    -Rajouter/supprimer les paramètres dans le dictionnaire self.args

Dans RK_ODE.py:
    -Dans step : changer le modèle
    -Dans ode_solution : Rajouter/supprimer les paramètres dans le même ordre que dans TKinter_visualization.py

Lancer TKinter_visualization.py

Pour lancer la simulation: appuyer sur "Entrer après avoir mesuré un paramètre"
